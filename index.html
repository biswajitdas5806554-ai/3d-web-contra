Visual style guide, simplified.

Retain the brand’s identity, but modernize it with depth, contrast, and motion that feels intentional.

✦

Color

Keep core brand tones, then add neon gradients and ambient glow for the 3D layers.

◌

Typography

Use strong headings, flat readable body text, and subtle depth only where it helps.

◎

Motion

Use soft easing, hover lift, and scroll reveals so the interface feels alive but calm.
